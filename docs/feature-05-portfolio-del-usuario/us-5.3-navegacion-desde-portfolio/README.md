# User Story 5.3 - Navegación desde portfolio

**Feature:** 5 - Portfolio del usuario

**Tipo:** Frontend

## Historia

**Como** usuario

**Quiero** navegar desde mi portfolio hacia valor actual, historial y watchlist

**Para** acceder rápidamente a las vistas principales de análisis y seguimiento.

## Criterios de aceptación

1. Dado un usuario en Portfolio Page, cuando selecciona valor actual, historial o watchlist, entonces navega a la pantalla correspondiente.
2. Dado un portfolio con posiciones, cuando entra a valor actual, entonces ve la valuación calculada usando el último precio disponible.
3. Dado un portfolio vacío, cuando entra a valor actual, historial o watchlist, entonces ve un estado vacío sin errores.
4. Dado operaciones existentes, cuando entra al historial, entonces se muestran ordenadas.
5. Dado una operación nueva, cuando vuelve al historial, entonces aparece reflejada.
6. Dado empresas agregadas a watchlist, cuando entra a la vista, entonces las ve listadas.
7. Dado una navegación desde mobile o web, cuando cambia de sección, entonces mantiene la sesión activa.
